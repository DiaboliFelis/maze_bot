# The Labyrinths Bot

**VK-бот для генерации и решения лабиринтов**

---

## Возможности

- 🏁 **Генерация лабиринтов** любого размера (от 2×2 до 50×50)
- 🧭 **Поиск кратчайшего пути** (алгоритм BFS)
- 🎨 **Визуализация** в виде PNG-картинок
- 💾 **Сохранение состояния диалога** после перезапуска
- ⌨️ **Управление кнопками** в ВКонтакте
